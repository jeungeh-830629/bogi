📺 bogi - 통합 멀티뷰 Streamlink GUI
bogi는 무거운 웹 브라우저 대신 가벼운 VLC 미디어 플레이어를 이용해 국내 주요 스트리밍 플랫폼의 라이브 방송을 최고 화질(Best)로 시청할 수 있게 해주는 파이썬(PyQt6) 기반 GUI 프로그램입니다.

✨ 주요 기능 (Features)
🎮 3대 플랫폼 완벽 지원: 치지직(CHZZK), 숲라이브(SOOP), 판다티비(PandaTV) 라이브 방송 지원.

⚡ 스마트 URL 캡처 (자동 낚아채기): 방송 썸네일에 마우스를 올리거나(SOOP) 클릭(치지직/판다티비)하는 즉시, 내부 자바스크립트 스파이가 작동하여 방송 주소를 하단 입력창으로 자동 추출합니다.

🖥️ 무제한 멀티뷰 (Multi-view): 추출된 주소로 ▶️ VLC 새 창으로 띄우기 버튼을 누를 때마다 독립된 백그라운드 스레드가 생성되어, 여러 개의 방송 창을 동시에 띄워놓고 시청할 수 있습니다. (창을 닫으면 스레드 자동 종료)

🔄 판다티비 풀방 자동 뚫기: "보유한 풀방입장권이 없습니다." 에러 발생 시, 5초 단위로 무한 재접속을 시도하여 빈자리를 자동으로 뚫어냅니다.

🔐 숲(SOOP) 로그인 및 비밀방 지원: 하단의 전용 입력창을 통해 SOOP 계정 로그인 및 비밀방 패스워드를 입력하여 시청할 수 있습니다. (자동 도메인 교정 기능 포함)

⚙️ 실행 조건 (Prerequisites)
PC에 **VLC 미디어 플레이어**가 설치되어 있어야 합니다. (프로그램이 레지스트리를 통해 설치 경로를 자동 탐색합니다.)

🛠️ 간편 빌드 방법 (How to Build)
파이썬 환경에서 단 두 번의 명령어만으로 실행 파일(.exe)을 빌드할 수 있습니다.

1. 필수 패키지 및 빌드 도구 설치

Bash
pip install PyQt6 PyQt6-WebEngine streamlink pyinstaller
2. PyInstaller를 이용한 프로그램 빌드

Bash
pyinstaller --noconsole --onedir --icon=bogi.ico --collect-all PyQt6 --collect-all PyQt6WebEngine bogi.py
*완료 후 생성된 dist/bogi 폴더 내부의 **bogi.exe*를 실행하시면 됩니다.

📖 간단 사용법 (Usage)
프로그램 상단에서 플랫폼(치지직, 숲, 판다티비)을 선택합니다.

내장 브라우저의 홈 화면에서 시청하고 싶은 방송 썸네일을 클릭하거나 마우스를 올립니다.

하단 입력창에 방송 주소가 자동으로 캡처되면 엔터 키 또는 [VLC 새 창으로 띄우기] 버튼을 누릅니다.

새로운 VLC 플레이어 창이 열리며 방송이 시작됩니다! (다른 방송을 추가로 클릭하여 계속 띄울 수 있습니다.)
